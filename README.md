## About

This is a minimal, elegant portfolio built with pure HTML, CSS, and JavaScript. It features:

- Clean, responsive design
- Project showcase with detailed descriptions
- Professional experience timeline
- Integrated contact form
- Smooth animations and transitions

## Built With

- HTML5
- CSS3 (Custom animations, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Work Sans)
- Google Forms integration

## Features

**Minimal Design** - Clean, distraction-free aesthetic focusing on content  
**Fully Responsive** - Optimized for all devices and screen sizes  
**Smooth Animations** - Elegant fade-ins and transitions  
**Project Showcase** - Featured projects with technologies used  
**Contact Form** - Google Forms integration with email notifications  
**Fast Loading** - No dependencies, pure HTML/CSS/JS  

## Sections

1. **Hero** - Introduction and call-to-action
2. **About** - Background, skills, and expertise
3. **Projects** - Featured projects in Machine Learning, Data Science, and Computer Vision
4. **Experience** - Professional internships and roles
5. **Contact** - Get in touch via integrated form
